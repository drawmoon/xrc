# XRC

XRC is a lightweight proxy tool with a command-line interface. It supports the VLESS, VMess and Trojan protocols.

## Features
- Manage remote subscription lists and filters
- Measure node latency and select the fastest nodes
- Run local HTTP and SOCKS proxies
- Integrates Xray protocol adapters (`vless`, `vmess`, `trojan`)
